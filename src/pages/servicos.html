Serviços e interesses

Principais serviços prestados e áreas de interesse para atuação e desenvolvimento.

Serviços

  • Automação de coleta de dados para processos operacionais.
    Desenvolvemos aplicações em python utilizando selenium ou node com puppeteer para automação de navegadores para coletar dados gerais, analisar informações e executar ações humanas como cliques e scroll de maneira automatizada.
  • Criação de bots para tarefas repetitivas e integração de rotinas.
    Fazemos bots para operar no Telegram para os mais variados fins, interação com o sistema operacional, agentes pessoais, monitoramento de outros programas e sistemas de pagamento e serviços automatizados.
  • Estruturação de informações para relatórios e acompanhamento de métricas.
    Usando bibliotecas como Streamlit produzimos dashboards e views para os mais variados tipos de dados, aliados a outros projetos como os scrappers e os bots de Telegram operando como um sistema gerencial completo.

Áreas de interesse

  • Desenvolvimento backend e integração de serviços para variados fins.
  • Blockchain, Criptografia, Segurança da informação e Privacidade.
  • Sistemas autônomos de monitoramento, raspagem, coleta e armazenamento de informações.
  • Desenvolvimento de software para finalidades educacionais, ambientais e sociais.
