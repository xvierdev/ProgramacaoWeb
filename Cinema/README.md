# Cinema

Aplicação para gerenciamento de catálogo de filmes desenvolvida em TypeScript.

## Requisitos

- Node.js (versão 18 ou superior)
- npm (gerenciador de pacotes do Node)

## Instalação

Para instalar as dependências do Node.js e do TypeScript, execute o comando:

```bash
npm install
```

## Compilação (Build)

Para converter o código TypeScript para JavaScript no diretório `dist`, utilize:

```bash
npm run build
```

## Execução

Existem duas formas de executar a aplicação:

### Modo de Desenvolvimento
Executa o projeto diretamente do código-fonte TypeScript:

```bash
npm run dev
```

### Modo de Produção
Executa o projeto a partir dos arquivos compilados no diretório `dist`:

```bash
npm start
```
